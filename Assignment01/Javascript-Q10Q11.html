Q11: Names: Store the names of a few of your friends in a array called names. Print each person’s name by accessing each element in the list, one at a time.

 

Solution

// Adding friends name in array
let friend_names = ['Owais Afsar','Yahya Hussaini','Mohammad Hammad','Maazuddin khan'];
// looping friend name from array
for(i=0; i < friend_names.length; i++) {
document.write(friend_names[i]); // Printing friend name
}
